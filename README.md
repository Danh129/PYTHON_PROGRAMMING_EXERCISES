## 📌 Giới thiệu tổng quan
Kho lưu trữ này là nơi mình tổng hợp các bài tập thực hành, thuật toán và dự án nhỏ trong quá trình học ngôn ngữ lập trình **Python** (Năm 2 - Học kỳ 1). 

Mục tiêu chính là nắm vững tư duy lập trình hiện đại, xử lý dữ liệu và xây dựng nền tảng cho các mảng chuyên sâu hơn như Web hoặc AI.

## 🏗 Công nghệ và Công cụ
* **Ngôn ngữ:** Python.
* **Thư viện thường dùng:** `math`, `datetime`, `os`, `sys`,... (có thể bổ sung `numpy`, `pandas` nếu có).
* **Công cụ:** Visual Studio Code, PyCharm hoặc Jupyter Notebook.

## 📂 Danh mục bài tập
* **`01_Kieu_Du_Lieu/`**: Biến, List, Tuple, Dictionary, String manipulation.
* **`02_Cau_Truc_Dieu_Khien/`**: Cấu trúc `if-elif-else`, vòng lặp `for`, `while` và xử lý ngoại lệ `try-except`.
* **`03_Ham_Module/`**: Định nghĩa hàm, Lambda function và cách import modules.
* **`04_OOP/`**: Lập trình hướng đối tượng (Class, Inheritance, Polymorphism).
